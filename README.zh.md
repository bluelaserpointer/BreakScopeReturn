# BreakScopeReturn 碎镜行动

展开镜子窥见盲区，反射子弹射击地形背后的敌人。

玩家将操控特工“棱”潜入盗取外星情报的超光速侦察装置“万花镜”的建造现场，阻止计划的施行。

本作品已完成20%。
